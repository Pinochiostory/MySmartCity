# MySmartCity

MySmartCity is a Django-based web application for reporting and tracking community service delivery issues in local areas. Users can report problems like water leaks, electricity faults, potholes, and waste collection issues, and track their progress.

## Features

- Report service delivery issues with details
- Track status of reported issues
- User-friendly interface based on conceptual design
- Supports different types of users (inclusive design)

## Requirements

- Python 3.13+
- Django 6.0.3
- Pillow 12.1.1
- tzdata 2025.3
- sqlparse 0.5.5
- asgiref 3.11.1

All dependencies are listed in `requirements.txt`.
